(async, static) obfuscateCID(privateCID, mnemonic8opt, saltopt) → {Promise.<Object>}

Description:

  • Obfuscates a high-entropy 256-bit CID into a human-memorable dual-list format. Uses Argon2id for key stretching and SHAKE256 for mask generation. The resulting mnemonic is expanded to 9 words, where the 9th word acts as an encoded checksum and shuffle-tail for recovery. Protects against "Evil Maid" attacks, perfect for handwritten backups.

Source:
  • cif.js, line 1204
Parameters:
Name Type Attributes Default Description
privateCID string

The 256-bit mnemonic or TV-code to obfuscate.
mnemonic8 string <optional>
 null

An optional 8-word mental key. If null, a new one is generated.
salt string <optional>
 'cif-defined-salt'

Secret lowercased salt for SHAKE256 and Argon2 derivation.
Throws:

If air-gap check fails or CID lacks sufficient entropy.

Type

Returns:
